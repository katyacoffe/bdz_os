1. В папке file хранится log.txt - результат моиторинга программы.
2. В папке obj  хранится Makefile, для запуска скрипта переходим в эту папку и прописываем make start
3. В папке src хранятся файлы программы, в которых:
  f1.cpp - Функция проверки возможности перехода на позицию (строка, столбец), от текущей позиции. Функция возвращает false, если (строка, столбец) недопустимая позиция или имеет значение 0 или уже посещено
  f2.cpp - Функция поиска кратчайший маршрут по методу ЛИ.
  f3.cpp - Функция мониторинга системы
  make.cpp - main файл
  в папке lib хранятся хэдэры.
4. Запуск програмы происходит из папки obj, для запуска прописываем "make start", далее вводим начальные координаты, затем конечные. На экране выводится кратчайший путь и результат мониторинга.
